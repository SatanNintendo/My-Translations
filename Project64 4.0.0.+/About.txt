Previously, this emulator had Russian language support, but the translation was outdated. Many phrases weren't quite correctly translated; I even found a spelling error (if I'm not mistaken).
Additionally, several new text strings were added to the emulator that simply displayed in English because they didn't exist at the time of the original translation.

I decided to remedy this situation and update the translation for one of the most respected emulators in the community. I wrote to the author about it but received no reply.
I also created a thread on GitHub, but it was closed after some time, with the Russian translation being replaced by Serbian for a reason unclear to me.

In any case, I don't know what's going through the project author's mind, but I'm forced to release the translation file in a separate branch.
It seems the emulator now supports fewer languages than it originally could, which is clearly a regression.

So, if you want to see the original Project64 in Russian, you need to download the "Russian.pj.Lang" file, then place it into the emulator's "Lang" folder.
That's all. You can then open the emulator and select the language.

Here's how: File -> Language -> Русский

That's it; you can now use the emulator with a Russian interface.
